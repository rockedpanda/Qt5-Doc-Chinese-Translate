Command Line Reader Async Example

The Command Line Reader Async example shows how to use the QSerialPort class for receiving data asynchronously over the selected serial port with the desired settings.

This command line reader async example receives data asynchronously over the selected serial port in a console, provided by the QSerialPort class.

For receiving data synchronously over the selected serial port, use the read() or readAll() methods, and readyRead() signal.
