Qt Quick Controls - Split View Example

An example for the SplitView UI control.

This example project demonstrates the usage of SplitView from Qt Quick Controls - a control that lays out items horizontally or vertically with a draggable splitter between each item.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, visit Building and Running an Example.
