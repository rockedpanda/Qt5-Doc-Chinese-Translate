Detailed Description

In addition to the normal Text element, Label follows the font and color scheme of the system. Use the text property to assign a text to the label. For other properties check Text.

A simple label looks like this:

Label {
    text: "Hello world"
}

You can use the properties of Text to change the appearance of the text as desired:

Label {
    text: "Hello world"
    font.pixelSize: 22
    font.italic: true
    color: "steelblue"
}

See also Text, TextField, and TextEdit.
