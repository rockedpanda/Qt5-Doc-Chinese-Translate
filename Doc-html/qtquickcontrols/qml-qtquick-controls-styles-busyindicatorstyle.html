BusyIndicatorStyle QML Type

Provides custom styling for BusyIndicatorStyle More...

Import Statement: import QtQuick.Controls.Styles 1.2
Since: Qt 5.2
  • List of all members, including inherited members

Properties

  • control : BusyIndicator
  • indicator : Component

Detailed Description

You can create a busy indicator by replacing the "indicator" delegate of the BusyIndicatorStyle with a custom design.

Example:

BusyIndicator {
    style: BusyIndicatorStyle {
        indicator: Image {
            visible: control.running
            source: "spinner.png"
            RotationAnimator on rotation {
                running: control.running
                loops: Animation.Infinite
                duration: 2000
                from: 0 ; to: 360
            }
        }
    }
}
