Qt Mac Extras Examples

Here is a list of example applications demonstrating the functionality provided by the Qt Mac Extras module.

Note: These examples work only on Mac OS X operating system.

Embedded QWindow

Demonstrates embedding a QWindow into an view hierarchy of a Mac OS X application.

Mac Functions

Demonstrates using functions from the QtMac namespace.

Mac Pasteboard MIME Converter

Demonstrates using QMacPasteboardMime class.
