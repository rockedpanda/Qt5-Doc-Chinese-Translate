Qt Mac Extras

Qt Mac Extras provide classes and functions that enable you to use miscellaneous functionality specific to Mac OS X and iOS operating systems.

Getting Started

To include the definitions of the module's classes and functions, use the following include statement:

#include <QtMacExtras>

To link against the module, add this line to your qmake .pro file:

QT += macextras
