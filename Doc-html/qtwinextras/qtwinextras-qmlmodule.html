Qt Windows Extras QML Types

QML Types

JumpListDestination

Represents a file that can be edited or viewed by an application.

JumpListLink

Represents a command to execute an application.

JumpListSeparator

Represents a task list separator.

DwmFeatures

Enables you to manage the Windows DWM features.

JumpList

Enables you to modify the application Jump List.

JumpListCategory

Represents a jump list category.

TaskbarButton

Manages Windows taskbar buttons.

ThumbnailToolBar

Allows manipulating the window's thumbnail toolbar.

ThumbnailToolButton

Represents a button in a thumbnail toolbar.
