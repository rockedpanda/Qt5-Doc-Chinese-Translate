Enginio Installation and Prerequisites

This guide shows how to use the Enginio Qt library in a Qt application (both Qt C++ and QML cases are covered). This guide can be also applied when integrating Enginio with existing Qt projects.

Prerequisites

Enginio comes with Qt 5.2 and later (for 5.2.0 only in the online installer). Make sure to select the Enginio component in the Qt Cloud Services category.

Note: Make sure to have OpenSSL installed for Enginio to work. On iOS, OpenSSL is not required as the native SSL framework is used.
