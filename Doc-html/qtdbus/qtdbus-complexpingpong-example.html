D-Bus Complex Ping Pong Example

Demonstrates usage of the Qt D-Bus typesystem.

Complex Ping Pong Example demonstrates the use of Qt D-Bus typesystem with QDBusVariant and QDBusReply. The example consists of the main application complexping which starts the other application, complexpong. Entering keywords such as hello and ping is handled by complexpong and the reply is printed to the standard output.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, visit Building and Running an Example.

To run, execute the complexping application.
