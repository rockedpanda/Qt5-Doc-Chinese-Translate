Detailed Description

The QMessageLogContext class provides additional information about a log message.

The class provides information about the source code location a qDebug(), qWarning(), qCritical() or qFatal() message was generated.

See also QMessageLogger, QtMessageHandler, and qInstallMessageHandler().
