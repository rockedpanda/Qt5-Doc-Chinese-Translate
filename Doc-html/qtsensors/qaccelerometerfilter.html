Detailed Description

The QAccelerometerFilter class is a convenience wrapper around QSensorFilter.

The only difference is that the filter() method features a pointer to QAccelerometerReading instead of QSensorReading.

Member Function Documentation

bool QAccelerometerFilter::filter(QAccelerometerReading * reading) [pure virtual]

Called when reading changes. Returns false to prevent the reading from propagating.

See also QSensorFilter::filter().
