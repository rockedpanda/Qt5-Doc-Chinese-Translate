SAX Bookmarks Example

Demonstrates how to read XBEL files

This example uses Qt's SAX API to read and parse the files. The DOM Bookmarks example provides an alternative way to read this type of file.

See the XML Bookmark Exchange Language Resource Page for more information about XBEL files.
