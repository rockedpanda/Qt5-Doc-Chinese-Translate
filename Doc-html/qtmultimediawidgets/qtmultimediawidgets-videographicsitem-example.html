Video Graphics Item Example

This example demonstrates how to stream video on a graphics scene.

The Video Graphics Item example shows how to implement a QGraphicsItem that displays video on a graphics scene using Qt Multimedia's QAbstractVideoSurface.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, visit Building and Running an Example.
