Building Qt Sources

There are two ways to install Qt:

  1. through the Qt Installers - downloads and installs Qt
  2. through the Qt sources.

You can download the Qt 5 installers and sources from the Downloads page. For more information, visit the Getting Started with Qt page.

Note: Qt Enterprise packages are available from the Try Qt Enterprise page.

This page lists the relevant information for installing Qt by building the Qt sources. The installation procedure is different on each Qt platform. This page collects the relevant information for the Supported Platforms.

OpenSSL (version 1.0.0 or later)

Support for Secure Sockets Layer (SSL) communication is provided by the OpenSSL Toolkit, which must be obtained separately.

From Qt version 5.2 onwards, the officially supported version for OpenSSL is 1.0.0 or later. Versions >= 0.9.7 and < 1.0.0 might work, but are not guaranteed to.

More information about enabling SSL support can be found in the Secure Sockets Layer (SSL) Classes document.
