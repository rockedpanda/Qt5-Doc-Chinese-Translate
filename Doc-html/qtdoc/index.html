Qt 5.3

Qt is a full development framework with tools designed to streamline the creation of applications and user interfaces for desktop, embedded, and mobile platforms.

Qt Framework

Intuitive APIs for C++ and JavaScript-like programming with Qt Quick for rapid UI creation.

Qt Creator IDE

A powerful cross-platform integrated development environment, including UI designer tools and on-device debugging.

Tools and Toolchains

Internationalization support, embedded toolchains, and more.
