All Namespaces

This is a list of the main namespaces in Qt.

Enginio

The Enginio namespace provides enums used throughout Enginio.

QAudio

The QAudio namespace contains enums used by the audio classes.

QBluetooth

The QBluetooth namespace provides classes and functions related to Bluetooth.

QDBus

The QDBus namespace contains miscellaneous identifiers used throughout the Qt D-Bus module.

QGL

The QGL namespace specifies miscellaneous identifiers used in the Qt OpenGL module.

QMediaMetaData

QMultimedia

The QMultimedia namespace contains miscellaneous identifiers used throughout the Qt Multimedia library.

QSql

The QSql namespace contains miscellaneous identifiers used throughout the Qt SQL module.

QSsl

The QSsl namespace declares enums common to all SSL classes in Qt Network.

QTest

The QTest namespace contains all the functions and declarations that are related to Qt Test.

QWebSocketProtocol

Contains constants related to the WebSocket standard.

Qt

The Qt namespace contains miscellaneous identifiers used throughout the Qt library.

QtAndroid

The QtAndroid namespace provides miscellaneous functions to aid Android development.

QtConcurrent

The QtConcurrent namespace provides high-level APIs that make it possible to write multi-threaded programs without using low-level threading primitives.

QtMac

The QtMac namespace contains miscellaneous functionality specific to Mac OS X and iOS operating systems.

QtWin

The QtWin namespace contains miscellaneous Windows-specific functions.
